<h1>Dungeon Delver</h1>
A basic text based dungeon crawler I produced for my Programming Fundamentals Module.

<h3>Aim</h3>
The players aim is to make their way through the dungeon fighting the monsters to upgrade your stats and finding more powerful weapons to kill the final boss to move on to the next level.
The game is infinite only ending on player death, the score tracked by coins.

<h3>Controls</h3>
The controls work using the number based menu provided using 1-x and pressing enter to confirm your choice.

<h3>Feature Overview</h3>
- Each level is randomly generated using a maze generator<br>
- Monsters move towards the best location near the player<br>
- An inventory the player can use using a series of text commands
