How to play the game
The game is a basic text-based dungeon crawler, the loop is as follows: search the dungeon for monsters and loot, fight the monsters to level up and increase your skills then find the boss of that level, kill the boss to move onto the next newly generated level.
Controls
Numbers are used mostly to select which options you’d like, when presented with an option input the number of the option you want, and press enter.
While using the inventory you can use words to carry out certain task’s “help” will bring up a list of commands you can use.
